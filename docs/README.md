# 📚 문서 디렉토리

이 디렉토리에는 프로젝트의 상세 문서들이 카테고리별로 정리되어 있습니다.

---

## 📁 디렉토리 구조

### 📦 `deployment/` - 배포 관련 문서
- `DEPLOYMENT_CHECKLIST.md` - 배포 전 확인 사항
- `GITHUB_DEPLOYMENT_GUIDE.md` - GitHub 배포 가이드
- `GITHUB_DEPLOYMENT_FINAL.md` - 배포 최종 확인
- `GITHUB_DEPLOYMENT_SUMMARY.md` - 배포 요약
- `PACKAGE_TEST_RESULTS.md` - 패키지 테스트 결과
- `COMMERCIAL_DEPLOYMENT_ANALYSIS.md` - 상업용 배포 분석

### 🏥 `medical/` - 의료 관련 문서
- `MEDICAL_READINESS_ANALYSIS.md` - 의료용 준비도 분석
- `CLINICAL_READINESS_CHECKLIST.md` - 임상 연구용 체크리스트
- `REGULATORY_MAPPING.md` - 규제 기준 매핑
- `COLD_HARD_TRUTH.md` - 냉정한 규제 기준 판정

### 📊 `analysis/` - 분석 문서
- `COMPLETENESS_ANALYSIS.md` - 완성도 종합 분석
- `CRITICAL_ARCHITECTURE_ANALYSIS.md` - 구조 파괴 및 재건축 계획
- `POTENTIAL_IMPROVEMENTS.md` - 최대 상승 가능 준비도 분석

### 🔄 `phase/` - Phase별 완료 보고서
- `PHASE1_COMPLETE.md` - Phase 1 완료 보고서
- `PHASE2_COMPLETE.md` - Phase 2 완료 보고서
- `PHASE3_COMPLETE.md` - Phase 3 완료 보고서
- `phase4_restructure_plan.py` - Phase 4 재구조화 계획

### 📖 `guides/` - 가이드 문서
- `EXTENSIBILITY_GUIDE.md` - 확장 가능성 가이드
- `IMPROVEMENTS_APPLIED.md` - 개선사항 보고서
- `UPDATE_SUMMARY.md` - 업데이트 요약
- `RUN_INSTRUCTIONS.md` - 실행 가이드
- `GITHUB_UPLOAD_GUIDE.md` - GitHub 업로드 가이드

## 📋 메인 디렉토리의 핵심 문서

메인 디렉토리에는 다음 핵심 문서만 있습니다:

- `README.md` - 프로젝트 개요 및 빠른 시작
- `INSTALLATION.md` - 설치 가이드
- `LEGAL_DISCLAIMER.md` - 면책 조항 (⚠️ 중요)
- `PHAM_BLOCKCHAIN_SIGNATURE.md` - 블록체인 서명
- `LICENSE` - MIT 라이선스
- `requirements.txt` - 의존성 목록
- `setup.py` / `pyproject.toml` - 패키지 설정
