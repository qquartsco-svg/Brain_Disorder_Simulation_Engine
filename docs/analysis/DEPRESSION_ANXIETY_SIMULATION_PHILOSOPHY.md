- 임상 의사결정 보조

### ✅ 이것임:

- **원인 분석 플랫폼**: 뇌 내부 동역학 이해
- **연구/교육 도구**: 인지적 패턴 관측
- **가설 검증 시스템**: 이론적 모델 테스트

## 🔬 우울증 시뮬레이션: 원인 분석 중심

### 핵심 질문들:

1. **"왜 부정적 편향이 발생하는가?"**
   - Amygdala의 과활성화 메커니즘
   - 부정적 자극에 대한 과민 반응
   - 긍정적 자극에 대한 둔감 반응

2. **"왜 인지 제어가 약화되는가?"**
   - PFC의 억제 기능 저하
   - 부정적 사고 루프 형성
   - 인지적 유연성 감소

3. **"왜 에너지가 고갈되는가?"**
   - Hypothalamus의 항상성 붕괴
   - 에너지 관리 시스템 이상
   - 수면-각성 리듬 혼란

4. **"왜 동기가 사라지는가?"**
   - Basal Ganglia의 보상 회로 이상
   - 도파민 시스템 저하
   - 목표 지향 행동 감소

### 시뮬레이션 목표:

```
입력 조건 → 뇌 내부 동역학 → 우울 패턴 관측
     ↓              ↓                    ↓
환경 스트레스    모듈 간 상호작용    부정적 편향
생물학적 요인    피드백 루프        인지 제어 약화
유전적 요인      신경전달물질 변화  에너지 고갈
```

**목적**: "어떤 조건에서 우울 패턴이 나타나는가?"를 탐색

## 😰 불안장애 시뮬레이션: 원인 분석 중심

### 핵심 질문들:

1. **"왜 위협 감지가 과민해지는가?"**
   - Amygdala의 위협 임계값 저하
   - 안전 신호와 위험 신호 구분 실패
   - 과각성 상태 유지

2. **"왜 감각 필터링이 실패하는가?"**
   - Thalamus의 게이팅 기능 약화
   - 관련 없는 자극까지 처리
   - 감각 과부하

3. **"왜 걱정 루프가 형성되는가?"**
   - PFC의 억제 제어 약화
   - 반복적 사고 패턴
   - 인지적 회피 실패

4. **"왜 스트레스 반응이 과활성화되는가?"**
   - Hypothalamus의 스트레스 시스템 과활성화
   - 자율신경계 불균형
   - 회복 메커니즘 약화

### 시뮬레이션 목표:

```
입력 조건 → 뇌 내부 동역학 → 불안 패턴 관측
     ↓              ↓                    ↓
위협 자극      Thalamus 필터링 실패    과각성
스트레스       Amygdala 과활성화      걱정 루프
과거 경험      PFC 억제 약화          회피 행동
```

**목적**: "어떤 조건에서 불안 패턴이 나타나는가?"를 탐색

## 🧠 Cookiie Brain Engine의 역할

### 메커니즘 탐색 도구로서:

1. **모듈별 독립적 분석**
   - 각 뇌 영역의 개별 기여도 측정
   - "이 모듈이 이상하면 어떤 패턴이 나타나는가?"

2. **상호작용 분석**
   - 모듈 간 피드백 루프
   - "이 연결이 끊어지면 어떻게 되는가?"
   - "이 루프가 강화되면 어떻게 되는가?"

3. **조건별 패턴 관측**
   - 다양한 입력 조건에서의 동역학 변화
   - "어떤 환경에서 이 패턴이 나타나는가?"

4. **시간적 변화 추적**
   - 패턴의 형성 과정
   - "어떻게 이 상태에 도달하는가?"

```python
    },

        "amygdala_pfc_loop": "negative_feedback_amplified",

        "social_support": "low",

    "reversibility": "low"        # 되돌리기 어려움
}
```

**해석**: "이러한 조건에서 이러한 메커니즘이 작동하여 우울 패턴이 나타남"

## ⚠️ 명확한 경계

### 시뮬레이터는:

✅ **할 수 있는 것**:
- 메커니즘 탐색
- 패턴 관측
- 가설 검증
- 이론적 모델 테스트

❌ **할 수 없는 것**:
- 실제 환자 진단
- 치료 효과 보장
- 임상 의사결정
- 의료 조언

### 용어 사용:

❌ **사용하지 않을 용어**:
- "진단"
- "치료"
- "회복"
- "개선"

✅ **사용할 용어**:
- "패턴 관측"
- "메커니즘 분석"
- "동역학 변화"
- "조건별 반응"

## 🔄 연구 흐름

### Phase 1: 원인 분석 (현재 단계)
```
질문: "왜 이런 패턴이 나타나는가?"
→ 시뮬레이션으로 메커니즘 탐색
→ 조건별 패턴 관측
→ 동역학 이해
```

### Phase 2: 가설 검증 (향후)
```
가설: "이 메커니즘이 이 패턴을 만든다"
→ 시뮬레이션으로 검증
→ 실험 설계
→ 데이터 비교
```

### Phase 3: 개입 효과 시뮬레이션 (연구용, 부차적)
```
질문: "이 조건을 바꾸면 패턴이 어떻게 변하는가?"
→ 파라미터 조정
→ 동역학 변화 관측
→ 메커니즘 이해 심화
```

**주의**: Phase 3도 "치료"가 아니라 "메커니즘 이해"가 목적
